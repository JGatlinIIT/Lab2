Cource Catalog Description:

This course will cover and assist in my learning of how Web pages are created, using HTML, CSS, Javascript and graphical applications. While also exploring networked multimedia distribution technologies. Along with that, we look at the design of effective Web site including page layout, user interface design, graphic design, content flow and site structure as well as management of Web site resources including intranet management and design considerations are addressed. We as students design and create a major Web site with multiple pages and cross-linked structures. Prerequisites: [none]Credit: 2-2-3 Semester Hours

Course Outcomes

We as students in this course will be able to:

  • Recognize HTML, CSS and JavaScript code in a web page/application.
  • Select and understand the proper mark-up tags or code to achieve a particular result.
  • Identify improperly used code.
  • Produce modern standards web pages.
  • Deploy web pages to a public server.

Course Objectives

At the conclusion of this course, each student will be able to

  • Explain the client and server architecture of the Internet and related web technologies.
  • Use a basic text editor and other software tools to create web pages using HTML, CSS, JavaScript, jQuery, and images.
  • Deploy web pages to a Linux-based web server using sftp.
